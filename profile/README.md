### Synapse-Talk는 대화형 AI 시스템을 처음부터 구축하는 프로젝트입니다.

최근 많은 AI 서비스는 이미 학습된 모델을 API 형태로 호출하여 사용합니다.  

하지만 실제 대화형 AI 시스템은 데이터 수집부터 모델 학습, 추론 서버, 서비스 제품화까지  
다양한 기술 요소가 결합된 복잡한 구조로 이루어져 있습니다.

Synapse-Talk는 이러한 구조를 직접 구현하며 **대화형 언어 모델이 실제로 어떻게 만들어지고 운영되는지**를  
엔지니어링 관점에서 이해하고 재현하는 것을 목표로 합니다.
